<!-- 一つ目キャリブレーションの話，二つ目バウンディングボックスの話？ -->
1. 本システムにおける，プロジェクタのキャリブレーションの流れを説明する．図に示すようにCGで構築された3次元モデル空間中にスクリーンと実環境のプロジェクタに相当するカメラを配置し，プロジェクタ視点からの映像を実環境中のプロジェクタから投影する．ここでCG空間における座標系を$\sum_M$とする．また，CG空間におけるプロジェクタとしてのカメラ$C_P$から観測されるシーンの画像座標系を$\sum_C_P$とする．
$\sum_M$における任意の点の3次元座標を$^{M}p$とし，$\sum_C_P$における画像座標をそれぞれ$^{C_p}m$とすると$^{C_p}m$はプロジェクタの内部パラメータA，回転行列R，並進ベクトルtからなる変換行列を用いて以下の式で表せる．\footnote(例えば$\boldsymbol{m}$は，座標mの同次表現を表し，"~"は定数倍を同値と見なす表現として用いている．)
\begin{equation}
{ }^{C_{p}} \tilde{\boldsymbol{m}} \sim \boldsymbol{A}[\boldsymbol{R} \boldsymbol{t}]^{M} \tilde{p}
\end{equation}
次に現実世界のプロジェクタを用いたスクリーンへの投影モデルを考える．実際の環境における世界座標系を$\sum_W$とし，スクリーン平面を$^W{z}=0$の平面と一致させると，スクリーン上に投影された点$^wp$はプロジェクタからスクリーンへの射影変換行列$H_{pj}$を用いて次の式のように表せる．
\begin{equation}
\left[{ }^{W} p_{x}{ }^{W} p_{y} 1\right]^{T} \sim \boldsymbol{H}_{\mathrm{pj}}{ }^{C_{p}} \tilde{\boldsymbol{m}}
\end{equation}
実際の外部カメラの画像座標系を$\sum_C$とすると，$^Wp$を実際のカメラで撮影した時の座標$^C_rm$はスクリーンからカメラへの射影変換行列$H_cm$と係数$s_3$を用いて次の式のように表せる．
\begin{equation}
{ }^{C_{r}} \tilde{\boldsymbol{m}} \sim \boldsymbol{H}_{\mathrm{cm}}\left[{ }^{W} \boldsymbol{p}_{x}{ }^{W} \boldsymbol{p}_{y} 1\right]^{\mathrm{T}}
\end{equation}
$H_pj$と$H_cm$は$\sum_C_P$と$\sum_C_r$の画像間の対応点から推定が可能であるため合成された射影変換行列を$H_{pj→cm}$とすると以下のように表すことが可能である．
\begin{equation}
{ }^{C_{r}} \tilde{\boldsymbol{m}} \sim \boldsymbol{H}_{\mathrm{pj} \rightarrow \mathrm{cm}}{ }^{C_{p}} \tilde{\boldsymbol{m}}
\end{equation}
式（1）を式（4）に代入し，定数sを用いると以下の関係式にまとめられる．
\begin{equation}
{ }^{C_{r}} \tilde{\boldsymbol{m}}=s \boldsymbol{H}_{\mathrm{pj} \rightarrow \mathrm{cm}} \boldsymbol{A}[\boldsymbol{R}, \boldsymbol{t}]^{M} \tilde{\boldsymbol{p}}
\end{equation}

以上の設定の下でCG空間においてプロジェクタの位置姿勢[R,t]を変化させることで，点$^Mp$を実環境におけるスクリーン上の投影の目標点$^{C_r}m$に位置ずれなく投影することを考える．
CG空間における複数の点$^MP_i$を対応する目標点$^{C_r}m_i$上に位置ずれなく投影するためにCG空間におけるプロジェクタの位置姿勢[$R^*,t^*$]を求める問題は最小二乗推定問題として以下の式で定式化可能である．
\begin{equation}
\left[\boldsymbol{R}^{*} \boldsymbol{t}^{*}\right]=\underset{\boldsymbol{R}, t}{\arg \min } \sum_{i \in S}\left\|^{C_{r}} \tilde{\boldsymbol{m}}_{i}^{*}-s \boldsymbol{H}_{\mathrm{pj} \rightarrow \mathrm{cm}, k} \boldsymbol{A}[\boldsymbol{R}, \boldsymbol{t}]^{M} \tilde{\boldsymbol{p}}_{i, k}\right\|^{2}
\end{equation}
ここでCG空間内でのプロジェクタの内部パラメータAは既知とし，sは設定する目標点のインデックス集合であり最低でも既知の6点を含む必要がある．
